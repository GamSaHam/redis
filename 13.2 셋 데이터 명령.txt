

sadd 키 요소[요소 ...]
시간 복잡도: O(1)
명령 설명: 인자로 입력된 요소를 주어진 키에 저장한다.
응답: <숫자응답> 성공이면 입력된 값의 수, 이미 존재하는 값이면 0

scard 키
시간복잡도: O(1)
명령 설명:주어진 키에 저장된 요소의 개수를 조회한다.
응답: <숫자응답> 저장된 요소의 개수

sdiff 키[키 ...]
시간 복잡도: O(N)
명령 설명: 주어진 키에 저장된 요소들의 차집합을 조회한다. 단 하나의 키만 입력되면 첫 번째 키에 저장된 모든 요소가 결과
로 조회된다.
응답: <멀티 벌크응답> 차집합에 해당하는 값 목록

sdiffstore 대상키 키[키 ...]
시간 복잡도: O(N)
명령 설명: 주어진 키에 저장된 요소들의 차집합을 조회하여 대상 키의 셋 데이터로 저장한다. 단, 하나의 키만
입력되면 첫 번째 키에 저장된 모든 요소가 대상키에 저장된다.
응답: <숫자응답> 저장된 값의 수

sinter 키[키 ...]
시간 복잡도: O(N)
명령 설명: 주어진 모든 키에 저장된 요소의 교집합을 조회한다. 단, 하나의 키만 입력되면 첫 번째 키에 저장된 모든 요소가
결과로 조회된다.
응답: <멀티 벌크응답> 차집합에 해당하는 값 목록

sinterstore 대상키 키[키 ...]
시간 복잡도: O(N^2)
명령 설명: 주어진 모든 키에 저장된 셋의 교집합을 조회하여 대상키에 셋 데이터로 조정한다. 단, 하나의 키만 입력되면 첫 번째
키에 저장된 모든 요소가 저장된다.
응답: <숫자응답> 저장된 값의 수

sismember 키 요소
시간 복잡도: O(1)
명령 설명: 입력된 요소가 주어진 키의 요소인지 확인한다.
응답: <숫자응답> 존재하면 1, 존재하지 않으면 0

smembers 키
시간 복잡도: O(N)
명령 설명: 주어진 키에 저장된 모든 요소를 조회한다.
응답: <멀티 벌크응답> 모든 값 목록

smove 원본키 대상키 요소
시간 복잡도: O(1)
명령 설명: 원본키에 저장된 요소를 대상키로 이동한다. 단, 대상키가 존재하지 않으면 새로운 키가 생성된다.
응답: <숫자응답> 이동되었으면 1, 실패하면 0

spop 키
시간 복잡도: O(1)
명령 설명: 주어진 키에 저장된 셋의 요소 중에서 임의의 요소를 조회하고 해당 요소를 제거한다.
응답: <벌크응답> 조회된 값

srandmember 키 [개수]
시간 복잡도: O(1)
명령 설명: 주어진 키에 저장된 요소 중에서 임의의 요소를 지정된 개수만큼 조회하고 해당 요소를 제거한다. 개수가
 입력되지 않으면 요소를 제거한다.
응답: <멀티 벌크응답> 조회된 값의 목록

srem 키 요소[요소 ...]
시간 복잡도: O(1)
명령 설명: 주어진 키에 저장된 요소 중에서 입력된 요소와 동일한 값을 가진 요소를 제거한다. 단, 주어진 요소가
존재하지 않으면 0을 돌려준다.
응답: <숫자응답> 제거된 값의 수

sunion 키[키 ...]
시간 복잡도: O(N)
명령 설명: 주어진 키에 저장된 요소들의 합집합을 조회한다. 단, 하나의 키만 입력되면 첫 번째 키에 저장된
모든 요소가 결과로 조회된다.
응답: <말티 벌크응답> 합집합에 해당하는 값의 목록

sunionstore 대상 키[키 ...]
시간 복잡도: O(N)
명령 설명: 주어진 키에 저장된 요소들의 합집합을 조회하여 대상키에 저장한다. 단, 하나의 키만 입력되면
첫 번째 키에 저장된 모든 요소가 대상키에 저장된다.
응답: <숫자응답> 저장된 값의 수
