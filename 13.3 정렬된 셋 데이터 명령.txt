

zadd 키 가중치 요소[가중치 요소 ...]
시간 복잡도: O(log(N))
명령 설명: 주어진 키에 입력된 요소와 가중치를 저장한다. 단, 주어진 요소가 이미 존재하면 저장된 요소의 가중치를
변경한다.
응답: <숫자응답> 새로 추가되었으면 1, 기존에 존재하면 0

zcard 키
시간 복잡도: O(1)
명령 설명: 주어진 키에 저장된 요소 수를 조회한다.
응답: <숫자응답> 존재하는 값의 수

zcount 키 시작가중치 종료가중치
시간 복잡도: O(log(N))
명령 설명: 주어진 키에 저장된 요소 중에서 시작가중치부터 종료가중치 범위에 해당하는 요소를 조회한다.
응답: <숫자응답> 조회된 값의 수

zincrby 키 값 요소
시간 복잡도: O(log(N))
명령 설명: 주어진 키에 저장된 요소중에서 지정된 요소의 가중치를 값만큼 증가시킨다.
응답: <숫자응답> 명령이 수행된 이 후의 가중치 값

zinterstore
시간 복잡도: O(N)
명령 설명: 주어진 저장된 요소의 교집합을 조회하여 대상키에 저장한다. 옵션을 입력하여 결과 셋에서 합계, 최소, 최
댓값을 조회할 수 있다.
응답: <숫자응답> 저장된 값 목록

zrange 키 시작인덱스 종료인덱스 [WITHSCORES]
시간 복잡도: O(log(N))
명령 설명: 주어진 키에 저장된 요소를 대상으로 시작인덱스부터 종료인덱스 범위에 해당하는 값들을 가중치 오름차순으로
조회한다.
응답: <숫자응답> 조회된 값 목록

zrangebyscore 키 시작가중치 종료가중치[WITHSCORES][LIMIT 오프셋 개수]
시간 복잡도: O(log(N))
명령 설명: 주어진 키에 저장된 요소를 대상으로 시작 가중치부터 종료 가중치 범위에 해당하는 값을 조회한다.
응답: <숫자응답> 조회된 값 목록

zrank 키 요소
시간 복잡도: O(log(N))
명령 설명: 주어진 키에 저장된 요소 중에서 주어진 요소의 순위를 조회한다.
응답: <숫자응답> 값이 존재하면 가중치 오름차순으로 정렬된 순위, 없으면 nil

zremrangebyrank 키 시작인덱스 종료인덱스
시간 복잡도: O(log(N))
명령 설명: 주어진 키에 저장된 요소 중에서 순위가 시작인덱스부터 종료인덱스 범위에 해당하는 요소를 삭제한다.
응답: <숫자응답> 삭제된 값의 수

zremrangebyscore 키 시작가중치 종료가중치
시간복잡도: O(log(N))
명령 설명: 주어진 키에 저장된 요소 중에서 시작가중치부터 종료가중치 범위에 해당하는 요소를 삭제한다.
응답: <숫자응답> 삭제된 값의 수

zrevrange 키 시작인덱스 종료인덱스 [WITHSCORES]
시간 복잡도: O(log(N))
명령 설명: 주어진 키에 저장된 요소 중에서 시작인덱스부터 종료인덱스 범위에 해당하는 요소를 점수기준 내림차순으로
조회한다.
응답: <숫자응답> 조회된 값 목록

zrevrangebyscore 키 종료가중치 시작가중치
시간 복잡도: O(log(N))
명령 설명: 주어진 키에 저장된 요소 중에서 시작가중치부터 종료가중치 범위에 해당하는 요소를 내림차순으로 조회한다.
시작가중치와 종료가중치의 입력순서에 유의하라
응답: <숫자응답> 조회된 값 목록, 결과 없을 때 nil

zrevrank 키 요소
시간 복잡도: O(log(N))
명령 설명: 주어진 키에 저장된 요소 중에서 주어진 요소의 내림차순 정렬순
